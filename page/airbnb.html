My Project

Airbnb Clone

Info of Project

  • Developed an interactive, web-based booking platform. User can post places and also can book the palces they like.
  • Front-end: Utilized React techniques and hooks for data display, employed Axios for server communication, and implemented Tailwind CSS for styling, achieving image uploading.
  • Back-end: Using express framework + mongoDB, using validator to verify input data.
Code repository: https://github.com/wenbo2978/Airbnb-Clone

Technique: React + Node.js + MongoDB + Express

Develop Tool: VSCode, Postman

Main Function

  • Login: login with username and password, if it is success, all information with user will store as cookie in brower, password will be encoded by JWT.
  • Register: register by basic user information, includes username and password, store them in database, password is encoded by JWT.
  • Add place: type basic place information by input or checkBox, also can upload image by link or from local computer, display preview of uploaded image. It also could delete image.
  • Edit place: edit information of place, include name, price, image...etc.
  • Booking place: display detail of place, user could book avaliable place with date and number of guests.
