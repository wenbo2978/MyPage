SahandEstate

Info of Project

  • Developed a real estate website designed for renting and selling properties, featuring secure user authentication, advanced search and filtering capabilities, and responsive design. The platform also enables users to send messages to property owners and receive notifications.
  • Leveraged React techniques for data display, integrated Redux Toolkit for local storage of user information, enabled Google account login through Firebase, and applied Tailwind CSS for styling.
  • Back-end: Developed with the Express framework and MongoDB, implemented JWT for user information protection, and utilized Firebase for cloud storage.

Code Repository: https://github.com/wenbo2978/mern-estate

Website: https://mern-estate-lid2.onrender.com/

Technique: React + Node.js + MongoDB + Express

Develop Tool: VSCode, Postman

Useful Tool: Firebase

Main Function

  • Login: (1) login with username and password, if it is success, all information with user will store as cookie in brower, password will be encoded by JWT. (2) login with google account, if it is not registered already, register with google account first.
  • Register: (1) register by basic user information, includes username and password, store them in database, password is encoded by JWT. (2) register by google account.
  • Create, edit, delete, listings
  • Search function: user could search by multiple conditions, includes name of place, rent or sell, providing parking lot, providinf wifi...etc. It also can sort by price or creating time.
  • Contact with landlord: if you are not the owner of the house, you can send message to the owner, and also, owner could reply it.
  • Notifications: any update or post new house will send notification to all users.
