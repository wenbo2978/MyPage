My Project

StaffTrainingSystem

Info of Project

  • Developed an interactive, web-based training platform. The platform allows administrators to manage user information, trainers to publish and review tasks, and staff to complete tasks.
  • Front-end: Implemented using JavaScript and jQuery, with AJAX for backend communication. Utilized Thymeleaf for dynamic data rendering in HTML pages.
  • Back-end: Built with Java and the Spring Boot framework.Employed MyBatis for MySQL database interactions.Conducted unit testing to ensure functionality and reliability.
Code repository: https://github.com/wenbo2978/StaffTrainingSystem

Technique: SpringBoot + MyBatis + MySQL + Thymeleaf

Develop Tool: Intellij IDEA, MYSQL Workbench

Main Function

Administrator

  • Manager users account: include create, edit and delete.

Trainer

  • Task: trainner can create task, edit and delete it. But when the trainner publish the task, any operation will be denied. And it will appear into staff's page.
  • Content of task: trainner can add multiple choice and essay for the existed task, edit and delete is also avaliable before the task is published.
  • Check processing: trainner can know the process of each staff.
  • Evaluate tasks: trainner need to grade tasks for each staff, include essay and multiple choice.
  • Sort: trainner could sort staff by socre, it could be the total socre or the score from one task.

Staff

  • Finish tasks: staff should finish task before deadline. They can do many times before they submit the taks, and any process will be saved.
  • Make feedback: staff could make feedback for each evaluation. And according to the feedback, trainner may grade again.
